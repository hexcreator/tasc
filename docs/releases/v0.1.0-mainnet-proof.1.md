# v0.1.0-mainnet-proof.1

This prerelease marks the first controlled Global Tasc Solana mainnet USDC payout proof.

It is not a public marketplace release, not audited, and not safe for unsupervised public funds. It is an evidence checkpoint for reviewers who want to inspect the end-to-end settlement loop and repeat it under a strict operator runbook.

## Known Gaps

- No external audit.
- No public beta wallet-extension QA by an independent operator.
- No deployed verifier operations.
- No dispute/reviewer market.
- No reputation, bonds, rate limits, or abuse controls.
- No decentralized indexer/verifier network yet.
